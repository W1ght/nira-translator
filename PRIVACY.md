# 隐私与数据说明

流译不包含账号系统，也不把翻译内容发送到项目作者控制的服务器。

当用户启用翻译时，以下内容会直接发送到用户所选模型服务：

- 被翻译的网页正文片段或选中文本；
- 目标语言、源语言和当前 Prompt；
- 用户选择的模型名称。

API Key 只用于从扩展后台向所配置的模型服务发起请求。它保存在浏览器本地的 `chrome.storage.local` 中，不会返回给网页 content script，也不会写入翻译缓存或日志。

`chrome.storage.local` 是本机持久化存储，并不等同于加密密钥库。能访问当前浏览器配置文件或设备的高权限软件仍可能读取其中的数据。建议：

- 使用专用于本扩展的项目 Key；
- 在服务商控制台设置消费上限；
- 定期轮换并撤销不再使用的 Key；
- 不要在共享设备上保存敏感生产密钥。

翻译缓存仅保存在 `chrome.storage.session`，并设置容量与条目上限。缓存键包含源文本的 SHA-256 摘要，缓存值只保存译文和必要元数据；浏览器会话结束、扩展更新或禁用后由浏览器清理。
